# pcb029v1


## 1. Общая информация

* Микроконтроллер: **stm32f103rct6**
* Внешний кварцевый резонатор: **8 MHz** (выводы PD0-OSC_IN и PD1-OSC_OUT1)
* BOOT0: **pull-down, 10k**



# 2. Внешние интерфейсы

## 2.1 USB Device

USB-интерфейс устройства реализован в режиме USB Full-Speed Device.

|Сигнал|Пин|Назначение|Примечание|
|---|---|---|---|
|USB_DN|PA11|USB Data Minus|Дифференциальная линия USB|
|USB_DP|PA12|USB Data Plus|Дифференциальная линия USB|
|DP_PULL|PA1|Подтяжка D+|Подключен к линии DP через резистор 1.5 кОм|
|VBUS_SENS|PA0|Детектирование VBUS|Подключен к шине питания через делитель напряжения|

## 2.2 UART1

* Назначение: соединение с хостом

|Сигнал|Пин|Функция|Примечание|
|---|---|---|---|
|TX1|PA9|UART1 Transmit|—|
|RX1|PA10|UART1 Receive|Внешняя подтяжка 10 кОм|


## 2.3 UART2

* Назначение: подключение внешних устройств **справа**

|Сигнал|Пин|Функция|Примечание|
|---|---|---|---|
|TX2|PA2|UART2 Transmit|—|
|RX2|PA3|UART2 Receive|Внешняя подтяжка 10 кОм|


## 2.4 UART3

* Назначение: подключение внешних устройств **слева**

|Сигнал|Пин|Функция|Примечание|
|---|---|---|---|
|TX3|PB10|UART3 Transmit|—|
|RX3|PB11|UART3 Receive|Внешняя подтяжка 10 кОм|


## 2.5 SWD (Debug Interface)

* Внешний разъем: **SWD-6-POGO2**

|Сигнал|Пин|Назначение|Примечание|
|---|---|---|---|
|SWDIO|PA13|Serial Wire Debug Data|Линия данных|
|SWCLK|PA14|Serial Wire Debug Clock|Тактовая линия|
