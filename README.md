# device05-fw-f103 — STM32F103 Firmware

## Архитектура

Проект построен как модульная embedded система:

### Слои
- Core: startup, system init, main loop
- BSP: абстракция конкретной платы
- Drivers: периферия (GPIO, UART, SPI, I2C)
- App: прикладная логика
- Middleware: утилиты

### Поток выполнения
main() → board_init() → app_init() → app_loop()

### Основные принципы
- Разделение ответственности
- Минимальная связанность слоёв
- Отсутствие прямого доступа App к регистрам STM32

### Структура
```
/Core
/Drivers/BSP
/Drivers
/App
/Middleware
/linker.ld
Makefile
```

### Core
Содержит точку входа, startup и системную инициализацию.

### BSP
Содержит реализацию под конкретную плату device05.

### App
Содержит бизнес-логику и сценарии работы устройства.

### Drivers
HAL/LL или кастомные драйверы периферии.
